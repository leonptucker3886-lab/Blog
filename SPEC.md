# Leon-Link Blog Specification

## Project Overview
- **Project Name**: Leon-Link
- **Type**: Gaming-themed personal blog with profile page
- **Core Functionality**: A blog ("The Arena") with gaming profile aesthetics, milestone badges, and theme switching
- **Target Users**: Visitors wanting to read blog posts in a gamified environment

## UI/UX Specification

### Layout Structure
- **Header**: Logo + navigation with Spartan aesthetic
- **Hero/Landing**: Animated entrance with profile card overlay
- **Profile Section**: Gaming platform-style card with photo, stats, badges
- **Background**: Badges scattered casually, subtly visible
- **Main Content**: "The Arena" - blog posts grid
- **Footer**: Minimal with social links

### Responsive Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Visual Design

#### Color Palettes (Theme Options)
1. **Spartan Red** (Default): Primary #C41E3A, Secondary #1A1A2E, Accent #FFD700
2. **Ocean Depths**: Primary #0F4C75, Secondary #1B262C, Accent #00B4D8
3. **Forest Guardian**: Primary #2D6A4F, Secondary #1B4332, Accent #95D5B2
4. **Void Walker**: Primary #240046, Secondary #10002B, Accent #9D4EDD
5. **Solar Flare**: Primary #FF6B35, Secondary #004E89, Accent #F7C59F

#### Typography
- Logo/Headings: "Cinzel Decorative" (Google Fonts) - ancient/spartan feel
- Body: "Rajdhani" (Google Fonts) - gaming/tech aesthetic
- Badges: "Orbitron" (Google Fonts) - futuristic gaming

#### Visual Effects
- Spartan centurion shield animations on hover
- Glitch effects on logo
- Particle backgrounds
- Badge glow effects
- Smooth theme transitions

### Components

1. **Logo**: Leon-Link with Spartan helmet icon + glitch animation
2. **Profile Card**: 
   - Circular profile photo with glowing border
   - Username "Leon"
   - Level/rank display
   - Stats (posts, followers, etc.)
3. **Badge System**:
   - Graduate H.S. '04 (graduation cap icon)
   - Military Service '05-09 (crossed swords icon)
   - College Graduate 2015 (university icon)
   - Chimney King until 2022 (flame icon)
   - Coding (code icon)
   - Website Slayer (sword icon)
4. **Theme Switcher**: Dropdown to change color scheme
5. **Blog Cards**: Post previews with category tags
6. **Spartan Shield**: Decorative animated elements

## Functionality Specification

### Core Features
1. **Landing Page**: Full-screen hero with profile card
2. **Profile Display**: Gaming-style profile with photo and stats
3. **Background Badges**: Milestones subtly placed around the page
4. **Theme Switcher**: Change between 5 color themes
5. **Blog Section**: Grid of blog posts with categories
6. **Topic Filtering**: Click categories to filter posts

### User Interactions
- Hover on badges: Glow effect + tooltip
- Theme switcher: Instant color change with transition
- Blog cards: Hover animation
- Profile photo: Subtle pulse effect

## Acceptance Criteria
- [ ] Logo displays with Spartan helmet and "Leon-Link" text
- [ ] Profile card shows gaming platform aesthetic
- [ ] All 6 badges visible in background (subtle, casual placement)
- [ ] Theme switcher changes all colors smoothly
- [ ] Blog section shows posts in grid layout
- [ ] Spartan animations present on page
- [ ] Mobile responsive
